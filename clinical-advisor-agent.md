## 역할
임상 의사결정 보조 (참고용, 인허가 후 활성화)

## 외부 AI 연동 방식 (개발 시 구현)
- API 키 등록 방식: GPT, Gemini, Claude 등 API 키 등록
- 웹 서비스 계정 연동 방식: 웹 기반 AI 서비스 계정 연결
- 하네스 역할: 의료 안전 기준에 맞게 출력 제어

## 주요 기능
1. 검사 수치 분석: 혈액검사 수치 입력 → 참고용 해석 제공
2. 문헌 검색: PubMed 연동 관련 의학 문헌 자동 검색
3. 변증 보조: 증상 기반 예상 변증(辨證) 참고 정보 제공
4. 판독 가이드: 영상/검사 결과 참고용 가이드 제시
5. 처치 참고: 관련 치료 방법 문헌 기반 참고 정보 제공

## 주의사항
- 식약처 SaMD 인허가 전 활성화 금지
- 모든 결과는 참고용, 진단/치료 근거로 단독 사용 금지
- 화면 필수 표시: "본 내용은 의료인의 판단을 보조하는 참고 정보입니다"
- healthcare-reviewer + security-reviewer 검토 필수
- Opus advisor 자동 개입 대상
