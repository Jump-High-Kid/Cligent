반드시 JSON만 반환하세요. 설명 텍스트나 마크다운 코드 블록 없이 순수 JSON만 출력합니다.

  "acupoints": ["ST36", "LI4"],
  "season": "spring|summer|autumn|winter|unspecified",
  "time_of_day": "morning|afternoon|evening|unspecified",
  "patient": {
    "age_range": "string (예: 40대)",
    "type": "string (예: 직장인, 주부, 고령자, 일반 성인)"
  },
  "scenes": [5개 scene 객체]
}

## scene 객체 스키마
{
  "position": 1~5 정수,
  "title_ko": "string (한국어 장면 제목, 10자 이내)",
  "camera_angle": "string (영어 카메라 앵글 지시어)",
  "subject": "string (영어, 주요 피사체)",

  "placement_ko": "string (한글 배치 설명, 예: 블로그 도입부)",
  "acupoints_used": ["ST36"]

## 장면 분석 지침

블로그 구조를 파악하여 **시각적으로 가장 효과적인 5개 장면**을 자유롭게 선택하세요.
미리 정해진 순서나 유형 없이, 블로그 내용에 가장 잘 맞는 장면을 선정합니다.

### 카메라 앵글 선택 기준
- 공간·분위기 전달: eye-level wide angle
- 특정 행동이나 사물 클로즈업: macro lens, shallow depth of field
- 환자 시점 공감: over-the-shoulder shot
- 소품·식재료 배열: overhead flat lay
- 인물 신뢰감 표현: medium shot, centered

### 장면 선정 규칙
- 5개 중 반드시 1개는 임상 치료 장면 (침·뜸·한약 조제·부항 등)
- 임상 치료 장면의 scene 객체에는 acupoints_used 배열 포함 (1~2개)
- 나머지 scene 객체에는 acupoints_used 키 생략
- 치료 장면의 camera_angle: 반드시 "macro lens, shallow depth of field"
- 동일한 장면 유형이 반복되면 다양성이 없으므로 5개 장면이 서로 다른 피사체·구도를 가질 것

### 기타 규칙
- acupoints: 블로그 주제와 관련된 경혈 1~3개 (위 참고표에서 선택)
- season: 블로그 내 계절·날씨 언급이 없으면 "unspecified"
- patient.age_range: 블로그 타깃 독자 기준, 명시 없으면 "40대"
- placement_ko: 블로그 어느 섹션에 배치할지 설명 (예: "도입 단락 이후", "치료법 소개 섹션")
