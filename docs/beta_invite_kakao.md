# 1차 베타 초대 — 카카오톡 발송용 메시지

> 운영자가 카카오톡으로 직접 발송할 때 사용하는 본문.
> 메일을 안 볼 가능성을 가정하고, 본문에 모든 핵심 정보 포함.

## 발송 절차
1. 어드민 패널에서 신청자 승인 → invite_url 생성 (72시간 만료)
2. 이 본문에서 `{invite_url}` 자리 실제 링크로 치환
3. 신청자 카카오톡으로 발송
4. 같은 신청자에게 E3 이메일도 자동 발송됨 (이중 채널)
