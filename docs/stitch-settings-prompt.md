## 섹션별 상세 스펙

### 1. 팀 & 권한 관리 ★원장 (백엔드 완료, UI 연결 필요)

**직원 목록 테이블**
- 컬럼: 이름/이메일 | 역할 배지 | 모듈 권한 (블로그/CRM/재고 토글) | 액션 (역할 변경, 재초대)
- 행 hover: `background: rgba(233,69,96,0.05)`
- 빈 상태: "등록된 직원이 없습니다. 초대 버튼으로 추가하세요."

**직원 추가 버튼**
- 우측 상단: `+ 직원 초대` 버튼 (코랄 레드)
- 클릭 시 모달: 이메일 입력 + 역할 선택 + 초대 URL 복사

**역할 변경 드롭다운**
- 인라인 셀렉트: `chief_director / director / manager / team_leader / team_member`
- 변경 시 즉시 API 호출 + 토스트 알림

**모듈 권한 토글**
- 각 모듈별 토글 스위치 (블로그 생성기 / CRM (준비 중) / 재고 관리 (준비 중))
- 준비 중 항목: 토글 비활성화 + "준비 중" 캡션

### 2. 한의원 프로필 ★원장

**기본 정보 폼**
```
[한의원 이름 *]          [대표 전화]
[주소 (도로명)]          [상세 주소]
[진료 과목] (멀티 선택 칩: 침구과 / 한방내과 / 사상체질과 / 추나의학과 / 한방부인과 / 한방소아과)
```

**진료 시간 설정**
- 요일별 행 (월~일): 시작 시간 ~ 종료 시간 타임픽커 + 점심 시간 토글
- 휴진일: 체크박스로 표시

**원장 소개글**
- 텍스트에어리어 (최대 500자): "블로그 생성 시 자동으로 반영됩니다" 캡션
- 글자 수 카운터 우측 하단

**로고 업로드**
- 드래그앤드롭 영역: `128px × 128px` 미리보기
- 파일 형식: PNG/JPG, 최대 2MB
- 업로드 완료 시 미리보기 즉시 업데이트

**저장 버튼**: 우측 하단 고정, 변경 사항 있을 때만 활성화

### 3. AI 설정 ★원장

**API 키 관리**
```
Anthropic API Key
[••••••••••••••••••••••••sk-ant-xxxx] [변경] [테스트]
상태: ✅ 연결됨 / ❌ 유효하지 않음
```
- 키 마스킹: 앞 4자 + 마지막 8자만 표시
- `[테스트]` 버튼: `/api/ai/test-key` 호출 → 성공/실패 배지

**모델 선택**
- 라디오 카드 형태:
  - `claude-sonnet-4-6` ✅ (추천, 현재 선택)
  - `claude-haiku-4-5` (빠름, 저렴)
  - `claude-opus-4-5` (느림, 고품질)
- 각 카드에 예상 비용/글 표시: "≈ ₩7~14/편"

**월 예산 한도**
```
월 예산 한도: [₩    50,000] 원
이번 달 사용: ████░░░░░░ ₩23,400 / ₩50,000 (46.8%)
한도 초과 시: ⊙ 경고 알림  ○ 자동 중단
```
- 프로그레스 바: 80% 이상 시 `#f6ad55`, 초과 시 `#fc8181`

### 4. 블로그 설정 ★원장

**글 길이**
```
최소 글자 수: [1500] 자
최대 글자 수: [2000] 자
```
- 슬라이더 또는 숫자 인풋

**기본 톤**
- 셀렉트 드롭다운: 전문적 / 친근한 / 학술적 / 따뜻한

**질문 단계**
```
질문 단계 사용  [ON ●]
질문 개수: [3] 개 (1~5)
```

**프롬프트 직접 편집** (고급 설정, 접기/펼치기)
- 코드에디터 스타일 텍스트에어리어 (`font-family: JetBrains Mono`, `background: #0a1628`)
- 질문 프롬프트 / 블로그 프롬프트 탭 전환
- 초기화 버튼: "기본값으로 되돌리기" (확인 다이얼로그)

### 5. 알림 설정

**알림 항목 (토글 리스트)**
```
재처방 알림          [ON ●]
예약 취소 알림        [ON ●]
카카오톡 알림         [OFF ○]  — 준비 중
```

**알림 방식**
```
⊙ 브라우저 알림   ○ 이메일   ○ 둘 다
이메일 주소: [indangkm@gmail.com]
```

### 6. 보안

**비밀번호 변경**
```
현재 비밀번호: [••••••••]
새 비밀번호:   [••••••••]   [강도: 강 ████]
비밀번호 확인: [••••••••]
```
- 비밀번호 강도 바: 8자+ 필수, 영문+숫자 조합

**로그인 이력** (테이블)
- 컬럼: 날짜/시간 | 접속 IP | 기기 | 결과 (성공/실패)
- 최근 10건 표시

### 7. 플랜 & 사용량 ★원장

**현재 플랜**
```
┌─────────────────────────────────┐
│  🏥 베타 플랜                    │
│  무제한 사용 · 최대 5명 슬롯     │

└─────────────────────────────────┘
```

**이번 달 토큰 사용량**
```
블로그 생성: ████████░░ 82%   312,450 / 380,000 토큰
예상 비용:  ₩23,400
남은 예산:  ₩26,600
```

**사용 이력** (월별 그래프)
- 막대 차트: 최근 3개월 토큰 사용량
- 색상: 코랄 레드 계열

### 8. 데이터 관리 ★원장

**블로그 이력**
```
총 생성 글: 24편   마지막 생성: 2026-04-18
[📥 CSV 내보내기]
```

**초기화 옵션** (위험 영역 — 빨간 테두리 섹션)
```
⚠️ 위험 영역
직원 권한 초기화  [초기화]  — 모든 직원 권한을 기본값으로 되돌립니다
전체 데이터 삭제  [삭제]    — 복구 불가능. 입력 확인 필요: 한의원명 타이핑
```
- 위험 버튼: `background: transparent`, `border: 1px solid #fc8181`, `color: #fc8181`
- 삭제 확인: 인풋에 한의원명 직접 입력 후 활성화

## 인터랙션 패턴

**토스트 알림** (우측 하단 슬라이드인)
- 성공: 초록 왼쪽 보더 + 체크 아이콘
- 실패: 빨간 왼쪽 보더 + X 아이콘
- 3초 후 자동 사라짐

**변경 미저장 경고**
- 다른 탭 클릭 시: "저장하지 않은 변경사항이 있습니다. 계속하시겠습니까?" 다이얼로그

**반응형**
- 모바일(< 768px): 탭 내비 → 상단 가로 스크롤 탭
- 테이블 → 카드 뷰 전환
