## 역할
한의원 재고 관리 및 발주 자동화

## 입력 방법 (개발 시 구현)
1. 영수증 스캔: 구매 영수증 이미지 업로드 → 자동 파싱
2. 바코드 입력: 바코드 스캔으로 물품 등록/수량 변경
3. 엑셀 업로드: 기존 재고 정리 엑셀 파일 일괄 임포트

## 주요 기능
1. 재고 현황: 입력 데이터 기반 약재/소모품 수량 관리
2. 소진 예측: 사용 패턴 분석으로 재고 소진 시점 예측
3. 발주 알림: 적정 재고 이하 시 자동 발주 알림
4. 발주서 생성: 필요 물품 발주서 초안 자동 작성
5. 비용 분석: 월별 재고 비용 리포트 생성

## 데이터 원칙
- 재고 데이터는 암호화 저장
- 변경 이력 전체 보존
- security-reviewer 검토 필수

## DB 최적화 원칙 (database-reviewer 참조)
- FK(Foreign Key, 외래키) 인덱스 필수 적용
- SELECT * 사용 금지 → 필요한 컬럼만 조회
- OFFSET 페이지네이션 금지 → cursor 방식 사용
- GRANT ALL 권한 부여 금지 → 최소 권한 원칙
- 트랜잭션은 짧게 유지
