한의원장의 시간을 돌려드립니다

블로그·마케팅·운영 — 한의원장의 진료 외 반복 업무를 Medical AI Agent, Cligent가 대신합니다.

베타 신청하기 어떻게 작동하나요?

신청 후 1~3일 내 회신 · 서비스 이용은 무료 · API 키만 본인 발급

문제

진료보다 진료 외 시간이 더 깁니다

한의원장이 직접 해야 하는 반복 업무. 자동화하지 않으면 매일 1~2시간씩 사라집니다.

1~2시간

블로그 1편 작성

주제 선정, SEO 키워드 분석, 이미지 첨부, 네이버 노출 최적화. 글 한 편에 들어가는 시간.

월 8~12편

네이버 SEO 유지

꾸준한 발행이 검색 순위의 핵심. 멈추는 순간 신환 유입이 줄어듭니다.

매월 반복

이미지·카드뉴스

블로그 한 편당 4~6장의 이미지가 필요. 프롬프트 직접 작성, AI 사이트 옮겨가며 생성.

파편화

분산된 도구

워드, 챗GPT, 미드저니, 네이버 블로그 — 결과물 하나당 도구 4~5개를 오갑니다.

솔루션

주제 입력 30초, 글 1편 완성

한의학 도메인에 특화된 프롬프트와 5레이어 패턴 엔진. 블로그·이미지 프롬프트를 한 흐름에서 끝냅니다.

STEP 1

주제만 입력

"불면증 한의학 치료" 한 줄. SEO 키워드는 자동 추출.

STEP 2

AI가 글 생성

30초 안에 1,500~2,000자, 네이버·구글 SEO 최적화. E-E-A-T 저자 byline 자동.

STEP 3

이미지 프롬프트

본문에 어울리는 이미지 프롬프트 4~6개 자동. Midjourney·ChatGPT·Gemini 클릭 한 번에 이동.

입력: 불면증 한의학 치료
옵션: 글자 수 2,000자 / 톤 친근함 / 패턴 ‘체계적 분석’
출력: 제목 → 서론 → 본론(원인/변증/처방) → 사례 → FAQ → 마무리 + 저자 byline
소요: 약 30~45초
BYOAI

당신의 API 키로, 토큰 비용은 0원

Bring Your Own AI — 일반 SaaS는 토큰 비용을 구독료에 얹습니다. Cligent는 그 부분이 없습니다. Anthropic Claude · OpenAI GPT · Google Gemini 모두 지원.

신뢰

한의원 데이터 처리에 PIPA·의료법을 그대로 반영. 환자 식별 정보는 입력 단계부터 차단.

개인정보보호법 22·26조, 의료법 19조 인용. 위탁업체(Anthropic 등)도 명시.

⏱

블로그 전문은 30일 후 서버에서 자동 삭제. 통계만 익명화 보관.

🔐

API 키 암호화

본인 API 키는 Fernet AES-256으로 암호화 저장. 평문 노출 없음.

🛡

환자정보 입력 차단

프롬프트 단계에서 환자 식별 정보 자동 차단. 의료법 위반 위험 0.

FAQ

자주 묻는 질문

API 키는 어디서 어떻게 받나요?
Anthropic Claude의 경우 console.anthropic.com에서 발급. 신용카드 등록 후 사용량 기반 결제. 평균 블로그 1편당 약 50~150원 (Claude 기준). 정식 영상 가이드는 베타 신청 후 함께 보내드립니다.
환자 정보를 입력해도 되나요?
안 됩니다. 의료법 19조(비밀유지 의무) 위반 가능성 때문에 환자 식별 정보(성명·생년월일·연락처·구체적 진료기록 등)는 프롬프트 단계에서 자동 차단됩니다. 일반적인 한의학 콘텐츠 작성에는 영향이 없습니다.
의료법에 위배되지 않나요?
Cligent는 진단·처방·치료를 대체하지 않습니다. 한의원장의 마케팅·운영 보조에만 사용되며, AI가 생성한 콘텐츠는 참고 자료입니다. 최종 검토와 게재 책임은 한의원장 본인에게 있습니다.
비용은 어떻게 발생하나요?
베타 기간엔 모든 기능 무료. 정식 출시 시 Standard 29,000원/월부터 시작 예정. 토큰 비용은 본인 API 키로 직접 결제하므로 Cligent 구독료에 포함되지 않습니다 (BYOAI).
한의원에 직원이 여러 명인데 같이 쓸 수 있나요?
베타 단계에선 한의원당 대표원장 1인만 가입 가능합니다. 직원 초대 기능은 정식 서비스 출시 이후 단계적으로 활성화됩니다.
해지 / 데이터 삭제는 어떻게 하나요?
언제든지 cligent.ai@gmail.com으로 요청하시면 즉시 처리합니다. 블로그 전문은 자동으로 30일 후 삭제되며, 회원 탈퇴 시 모든 개인정보가 삭제됩니다.
