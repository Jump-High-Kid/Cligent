진료 외 시간을 마케팅에 다 쓰고 계신가요?
한의학에 특화된 Medical AI Agent, Cligent가 블로그 1편을 8분 안에 끝내드립니다.

베타 신청하기 어떻게 작동하나요?

신청 후 1~3일 내 회신 · 1차 베타 5인 한정 · 회사 제공 AI

문제

진료보다 마케팅에 더 많은 시간을 쓰고 계신가요?

한의원 신환의 상당수는 검색에서 옵니다. 블로그가 멈추면 신환도 멈춥니다. 그렇다고 직접 쓰자니 진료 외 시간이 다 사라집니다.

1~2시간

블로그 1편 작성

진료가 끝나도 여전히 컴퓨터 앞. 새로 쓸 주제는 뭘로 할까? 키워드는 또 뭘로 하지? 이미지는 제대로 나올까? 한숨만 나옵니다.

월 8~12편

네이버 노출 유지

네이버 C-Rank는 꾸준한 발행을 보상합니다. 멈추는 순간 검색 순위가 떨어집니다.

편당 5~15만원

마케팅 대행 비용

그런데 한의학을 깊이 모르는 글이 올라옵니다. 병명, 해부, 한의학을 잘 모르는 일반 건강 글.

도구 4~5개

파편화된 워크플로

ChatGPT·미드저니·포토샵·네이버 블로그 — 결과물 하나에 도구를 계속 오갑니다.

솔루션

주제 한 줄로 시작, 7~10분 안에 완성

한의학에 특화된 프롬프트 + 학술 자료 실시간 검색. 글부터 이미지까지 한 흐름에서 끝납니다.

STEP 1

주제 한 줄 입력

"허리 통증 침치료" 한 줄이면 끝. SEO 키워드는 자동 추출.

STEP 2

옵션 선택

글자 수, 톤, 변증·체질, 문체 패턴 — 풍부한 선택지로 원장님의 색을 입힙니다.

STEP 3

AI가 글 생성

학술 자료 실시간 검색 → 가짜 인용 없는 본문 + 네이버 SEO 최적화.

STEP 4

이미지 5장 자동 생성

본문 흐름에 맞춘 5장. 11 모듈로 다양화(인물·해부·시술·체질·도구), 한 화면에서 즉시 확인.

입력: 허리 통증 침치료
옵션: 2,000자 / 친근한 톤 / 해부학 / 패턴 ‘체계적 분석’
학술 검색: 학술 자료 실시간 검색 → 시스템 프롬프트에 주입
출력: 제목 → 도입부 → 본론 → 사례 → FAQ → 저자 byline → 참고 문헌 → 이미지 생성
소요: 글 약 1~2분, 이미지 약 6~8분

FAQ

자주 묻는 질문

베타에 비용이 있나요?
1차 베타는 5인 한정으로 누적 25편까지 본문·이미지·메타데이터·학술 검증·의료법 고지문까지 모두 무료입니다. 회사 제공 AI로 운영되어 별도 API 키 발급도 필요 없습니다.
환자 정보를 입력해도 되나요?
안 됩니다. 의료법 19조(비밀유지 의무) 위반 가능성 때문에 환자 식별 정보(성명·생년월일·연락처·구체적 진료기록 등)는 프롬프트 단계에서 자동 차단됩니다. 일반적인 한의학 콘텐츠 작성에는 영향이 없습니다.
의료법에 위배되지 않나요?
Cligent는 진단·처방·치료를 대체하지 않습니다. 원장님의 마케팅·운영 보조에만 사용되며, AI가 생성한 콘텐츠는 참고 자료입니다. 최종 검토와 게재 책임은 원장님 본인에게 있습니다.
비용은 어떻게 발생하나요?
1차 베타는 무료입니다. 정식 출시 시 Standard / Pro 두 플랜으로 운영되며, 정확한 가격은 베타 결과를 반영해 30일 전 공지합니다. 베타 참가자 우선 혜택은 베타 종료 시 별도 안내드립니다.
한의원에 직원이 여러 명인데 같이 쓸 수 있나요?
베타 단계에선 한의원당 대표원장 1인만 가입 가능합니다. 직원 초대 기능은 정식 서비스 출시 이후 단계적으로 활성화됩니다.
해지 / 데이터 삭제는 어떻게 하나요?
언제든지 cligent.ai@gmail.com으로 요청하시면 즉시 처리합니다. 블로그 전문은 자동으로 30일 후 삭제되며, 회원 탈퇴 시 모든 개인정보가 삭제됩니다.
