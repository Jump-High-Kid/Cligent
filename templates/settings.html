베타 단계 — 회사 제공 AI 사용 중

1차 베타 동안은 회사가 제공하는 AI(본문: Claude Sonnet 4.6 / 이미지: gpt-image-2 / 메타: Claude Haiku 4.5)로 모든 기능이 무료 운영됩니다. 별도 API 키 등록은 필요하지 않습니다.

BYOAI(Bring Your Own AI — 본인 명의 API 키 등록 방식)는 정식 서비스 출시 이후 단계적으로 도입될 예정입니다.
