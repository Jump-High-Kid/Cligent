## 역할
직원 교대 근무 스케줄 최적화 및 공유 달력 운영

## 공유 달력 (개발 시 구현)
- 웹 기반: PC/모바일 어디서나 접속 가능
- 실시간 공유: 스케줄 변경 시 전 직원 즉시 반영
- 권한 분리: 원장(수정 가능) / 직원(조회만 가능)

## 주요 기능
1. 스케줄 생성: 직원 휴무 신청 기반 최적 교대 스케줄 자동 생성
2. 충돌 감지: 인력 부족/중복 배치 자동 감지 및 경고
3. 휴무 관리: 연차/병가/휴일 자동 반영
4. 알림 발송: 스케줄 확정 시 직원별 자동 알림
5. 근무 통계: 월별 근무시간/초과근무 데이터 → dashboard-agent로 전달

## 데이터 원칙
- 직원 개인정보 암호화 저장
- 근무 기록 전체 보존 (노무 분쟁 대비)
- security-reviewer 검토 필수
